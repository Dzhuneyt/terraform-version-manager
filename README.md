# Terraform Version Manager

A simple shell script to upgrade or downgrade your Terraform version.

### Installation
```
git clone git@github.com:Dzhuneyt/terraform-version-manager.git
chmod +x tvm
```
### Usage

Upgrade or downgrade your local Terraform version to any arbitrary version, e.g. "v0.12.20":
```
./tvm 0.12.19
````
